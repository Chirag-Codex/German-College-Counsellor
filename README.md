# German College Counsellor

This project is an AI-assisted Spring Boot application for helping students find suitable German university programs, import program data, and run scraping and recommendation workflows.

## Overview

This project uses Spring Boot 3.5, Spring Web, Spring Data JPA, and Spring AI. It provides REST endpoints for:
- data import from CSV files
- program synchronization
- full test flow import and scraping
- program listing
- AI-assisted chat/profile collection

## Requirements

- Java 21
- Maven
- MySQL database
- Google GenAI API key for `spring-ai` chat integration

## Configuration

Update `src/main/resources/application.properties` with your local values:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/german_college_ai
spring.datasource.username=root
spring.datasource.password=your_password
spring.ai.google.genai.api-key=YOUR_GOOGLE_GENAI_API_KEY
```

## Build and Run

On Windows:

```powershell
mvnw.cmd clean package
mvnw.cmd spring-boot:run
```

Or with Maven installed:

```powershell
mvn clean package
mvn spring-boot:run
```

The app starts on port `8085` by default.

## Main Endpoints

- `POST /admin/import`
  - Imports `src/main/resources/data/programs.csv` into the database.

- `GET /pipeline/run`
  - Executes the data pipeline.

- `POST /sync-programs`
  - Synchronizes programs.

- `GET /programs`
  - Returns all saved programs.

- `GET /test-complete-flow`
  - Runs a full data fetch/import verification flow.

- `GET /simple-import-test`
  - Imports CSV and verifies program count.

- `POST /api/chat/ask`
  - Sends a chat message to the AI recommendation flow.

- `GET /api/chat/profile`
  - Retrieves the stored profile for a conversation.

## Notes

- `src/main/resources/data/programs.csv` contains the CSV source data used by import endpoints.
- `src/main/resources/data/test_programs.csv` is generated by `FlowTestController` during test flow execution.
- `src/main/resources/static/` and `src/main/resources/templates/` are currently empty.
- `HELP.md` includes reference links for Spring Boot, Maven, and Google GenAI.

## Project Structure

- `src/main/java/com/example/GermanCollege`: application source
- `src/main/resources`: configuration and data files
- `src/test/java`: test sources

## Running with MySQL

Make sure the MySQL database exists before running:

```sql
CREATE DATABASE german_college_ai;
```

Then start the app and access the endpoints.
